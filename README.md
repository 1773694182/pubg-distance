# PUBG 测距工具

一个 Python Tkinter 实现的 PUBG 地图测距工具，支持 2K、4K 分辨率，支持 8x8/4x4/2x2 地图网格比例估算。工具支持全局鼠标取点，不需要在工具窗口内点击。

## 功能

- 分辨率选择：`1920x1080`、`2560x1440`、`3840x2160`
- 地图模式：`8x8` 大地图、`4x4` 地图、`2x2` 地图
- 取点按键可选择：鼠标侧键、鼠标右键、鼠标左键、鼠标中键、自定义按键
- `F2` 或 `Esc` 重新计算并清空已选点
- 标记两个点后，在屏幕左上角独立黄色浮层高亮显示估算米数
- 使用 `pynput` 全局监听和 Windows `GetAsyncKeyState` 轮询监听，提高游戏内取点成功率
- 分辨率、地图模式和取点按键设置会自动记住，下次启动恢复

## 使用

```bash
python pubg_distance_tool.py
```

在界面中选择当前分辨率、地图模式和取点按键，然后把鼠标移动到地图上的目标位置，用选定按键标记两个点。

快捷键和鼠标操作：

- 鼠标侧键 / 左键 / 中键 / 右键 / 自定义按键：按界面选择标记当前鼠标位置
- 自定义按键：点击「设置按键」后按任意键（支持 F1-F12、PageUp、方向键等），按键即被捕获用作取点
- 鼠标滚轮向上：等比放大地图比例
- 鼠标滚轮向下：等比缩小地图比例
- `F2` 或 `Esc`：清空已选点，重新计算

地图模式只能在操作界面中选择，键盘快捷键不会切换 8x8/4x4/2x2。

滚轮缩放会改变当前 `px/100m` 比例，并立即影响距离计算。默认最大值按屏幕宽度等比换算，4x4 和 2x2 使用单独校准值。4x4：4K 为 `54px/100m` 到 `850px/100m`，2K 为 `22.9px/100m` 到 `360px/100m`。2x2：4K 为 `108px/100m` 到 `1730px/100m`，2K 为 `72px/100m` 到 `1153.3px/100m`。切换分辨率或地图模式时会重置为对应基础比例。

如果在游戏内无法监听按键或侧键，或者左上角浮层被游戏覆盖，请以管理员身份运行 `PUBGDistanceTool-Global.exe`，并使用无边框窗口化模式运行游戏。打包脚本默认添加 `--uac-admin`，新版 exe 启动时会请求管理员权限。

如果换到另一台没有 Python 环境的电脑，直接运行打包后的 exe 即可，不需要安装 Python。请确认复制的是重新打包后的 `dist\PUBGDistanceTool-Global.exe`。

## 打包 EXE

Windows 下双击运行：

```bat
build_exe.bat
```

打包完成后文件位于：

```text
dist\PUBGDistanceTool-Global.exe
```

也可以手动执行：

```bash
py -3 -m pip install -r requirements.txt
py -3 -m PyInstaller --onefile --windowed --uac-admin --name PUBGDistanceTool-Global pubg_distance_tool.py
```

## 比例说明

地图基础比例以 8x8 网格为基准；4x4 和 2x2 在 2K/4K 下使用校准值：

- 1080p：8x8 约 `38.4px`，4x4 约 `153.6px`，2x2 约 `614.4px`
- 2K：8x8 约 `18px`，4x4 约 `22.9px`，2x2 约 `72px`
- 4K：8x8 约 `27px`，4x4 约 `54px`，2x2 约 `108px`
